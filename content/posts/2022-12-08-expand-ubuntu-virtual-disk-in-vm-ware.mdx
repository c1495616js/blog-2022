I use Ubuntu in VMWare on my windows machine.
This post is to talk about how to expand the disk space on Ubuntu in VMWare.

## Steps:

1. Shut down the Ubuntu.
2. Open the settings and click on the `expand` button.
   ![](/images/2022-12-08-expand-ubuntu-virtual-disk-in-vm-ware/2022-12-08-09-49-35.png)

3. Now go back to Ubuntu.

4. Install the package `gparted`.

```shell
$ sudo apt install gparted
```

5. open `gparted`

```shell
$ gparted
```

6. select ` sda2` and `sda5`, right click on `Resize/Move`,
   and drag the space on the top to full and then apply the change.
   ![](/images/2022-12-08-expand-ubuntu-virtual-disk-in-vm-ware/2022-12-08-09-53-38.png)
